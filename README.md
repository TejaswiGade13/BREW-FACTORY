# ☕ Brew Factory

A modern coffee house landing page featuring smooth scrolling, immersive animations, and an engaging user experience built with HTML, CSS, and JavaScript.

<p align="center">
  <img src="assets/demo.gif" alt="Brew Factory Demo" width="1000">
</p>

---

## ✨ Features

- ☕ Elegant coffee house landing page
- 🎥 Hero section with a background video
- 🚀 Smooth scrolling powered by **Locomotive Scroll**
- ✨ Scroll-triggered animations using **GSAP** and **ScrollTrigger**
- 🔤 Animated typography with **SplitType**
- 🛍️ Interactive product showcase
- ⭐ Customer testimonials section
- 🎨 Clean and modern UI
- 📂 Organized project structure
